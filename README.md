# 华硕改版固件Asuswrt-Modx
MT7621AT华硕改版固件支持软件中心

## 支持的设备
- 小米R3G
- 极路由4增强版
- 中兴E8820S
- 诺基亚A-040W-Q（闲鱼卖家赞助）
- 新路由3（需要拆除CPU附近的电容C48）
- 新路由2
- 小娱C3
- 京东云1代（群友赞助）
- 瑞斯达康MSG1500（群友赞助）
- 小米R3P
- 红米/小米AC2100（群友赞助红米AC2100）

## 如何编译固件
1. cd release/src-ra-5010
2. make model (currently available models are: rt-mir3g, rt-mir4a, rt-rm2100, rt-r2100)

## 不支持的功能
- 双wan
- 无线中继（纯MT7615机型已支持）
